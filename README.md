# MLBB Esports Predictor

A machine learning project focused on predicting match outcomes in Mobile Legends: Bang Bang professional esports matches using in-game statistics.

## Current Progress

- Collected match statistics manually from tournament PDFs
- Extracted raw match data using Python
- Converted extracted data into CSV dataset
- Performed exploratory data analysis

## Tech Stack

- Python
- Pandas
- Jupyter Notebook

## Future Plans

- Expand dataset size
- Feature engineering
- Train machine learning model
- Build win probability predictor
